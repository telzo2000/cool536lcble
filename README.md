# cool536lcble

## What is cool536lcble?

The cool536lc key layout has been revised to create a 36-key keyboard that can be connected wirelessly via Bluetooth.

<br>

![](img/img00001.jpg)

## firmware

 cool536lc run QMK_firmware,use BLE Micro Pro.
